# NinaUnect
A software for presence management.

## Processo de Desenvolvimento
``` bash
# Clone o repositório
git clone https://github.com/ViniciusRomano/NinaUnect

# Dependências
pip install requests

# Rodando o programa (Python 2.7)
python tk.py

# Nova branch com a funcionalidade que o dev vai desenvolver. Ex: criando_crud

git checkout -b criando_crud

#Commit na sua feature funcionando
git add .
git commit -m 'criado o crud da aplicação'
git push origin criando_crud

# Abra um pull request explicando sua funcionalidade e espere outro dev dar merge pra versão estavel.
# Caso queria continuar desenvolvendo enquanto espera o pull request ser fechado crie outra branch.
```
